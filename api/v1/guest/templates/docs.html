Introduction

v1.0.0

LogArbor is a modern log aggregation and monitoring platform designed for developers who want real-time insight into their services without complexity.

Why LogArbor?

  • Terminal-style real-time log streaming
  • Simple API & CLI
  • Service-level isolation
  • Developer-first UX

Quick Example


# Install CLI
pip install logarbor

# Initialize a service
logarbor init --service auth-service

# Send a log
logarbor send "User logged in" --level info
