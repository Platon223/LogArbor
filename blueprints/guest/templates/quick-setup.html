Quick Setup

LogArbor is designed to get you logging in under 5 minutes. Follow the steps below to start sending logs immediately.

1. Create a Service

Each application must belong to a service. Services isolate logs, API keys, and permissions. 

logarbor service create auth-service

2. Configure Environment

Add your API key to your environment variables:

export LOGARBOR_API_KEY=your_api_key_here
        export LOGARBOR_SERVICE=auth-service

3. Send Your First Log

logarbor send "Service started successfully" --level info
